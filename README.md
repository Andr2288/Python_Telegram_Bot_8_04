# Telegram-бот нагадувань

Бот для створення нагадувань, сповіщень у вказаний час, списків, пошуку та статистики. Дані зберігаються в **SQLite**. Стек: **Python**, **python-telegram-bot** (з `job-queue`), **zoneinfo**.

Повне ТЗ: файл `Вимоги до боту.txt` у корені проєкту. Дорожня карта та handoff для розробників: `PLAN.md`.

## Запуск

1. **Python** 3.11+ (перевірено на 3.12–3.14).
2. Створити віртуальне середовище і встановити залежності:

   ```text
   python -m venv .venv
   .venv\Scripts\activate
   pip install -r requirements.txt
   ```

   На Linux/macOS: `source .venv/bin/activate`.

3. Скопіювати `.env.example` у `.env` і вказати токен від [@BotFather](https://t.me/BotFather):

   ```env
   BOT_TOKEN=123456789:ABC...
   ```

4. Запуск бота:

   ```text
   python main.py
   ```

Після першого запуску з’явиться файл `reminders.db` у корені проєкту.

## Змінні середовища

| Змінна      | Опис                          |
|------------|-------------------------------|
| `BOT_TOKEN` | Токен бота Telegram (обов’язково) |

## Структура проєкту

| Шлях            | Призначення                                      |
|-----------------|--------------------------------------------------|
| `main.py`       | Точка входу, реєстрація handlers, `run_polling`  |
| `config.py`     | Завантаження `BOT_TOKEN` з `.env`                |
| `database/`     | SQLite: користувачі, нагадування, `activity_log` |
| `handlers/`     | Команди та `ConversationHandler` для /add, /edit  |
| `jobs/`         | JobQueue: відправка нагадувань у потрібний час   |
| `helpers/`      | Парсинг дат/часу, «нагадай …», повтори, TZ       |
| `PLAN.md`       | План робіт і поточний стан функцій               |

## Приклади команд

- `/start`, `/help` — привітання та довідка  
- `/add` — структуроване створення (текст → дата → час); `/cancel` — вийти з кроків  
- `/list`, `/history`, `/edit`, `/delete <id>`  
- `/timezone` та `/timezone Europe/Kyiv`  
- `/search молоко`, `/search нарада active`  
- `/stats`  

Також у **приватному чаті** можна написати, наприклад:  
`нагадай завтра о 9 купити молоко`, `нагадай щодня о 8 зарядка`.

## Примітки

- Одночасно має працювати лише **один** процес бота з тим самим токеном (інакше Telegram повертає `409 Conflict`).  
- На Python 3.12+ у `main.py` уже налаштовано створення event loop для сумісності з `run_polling`.
